## Features

- Windows 10-style desktop, taskbar, start menu, and draggable windows
- About Me window with Lanyard / Discord presence
- Spotify media player powered by Lanyard
- GitHub repositories and contribution graph
- Friends folder with custom icons
- Active Projects window with project drill-in info
- Dark mode Windows-style UI

## Local Development

```bash
node server.mjs
```

Then open:

```text
http://127.0.0.1:4173/
```
